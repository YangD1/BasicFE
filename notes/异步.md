## 单线程和异步
- JS 是**单线程**语言，只能同时做一件事
- 浏览器和 NodeJS 已支持 JS 启动进程（Web Worker），不意味着改变了JS单线程语言的本质
- JS 和 DOM 渲染共用同一个线程（DOM渲染和JS执行同时只有一个可以工作），因为 JS 可修改 DOM 结构
> 异步的背景就是根据JS单线程的特点来的  
异步为了解决遇到等待（网络请求，定时任务）不能卡住  
异步基于回调 callback 函数形式  

## 异步和同步的区别
- 基于 JS 是单线程语言
- 异步**不会**阻塞代码执行
- 同步**会**阻塞代码执行
